Installation
==================

Clone the development version:

.. code-block:: bash

    git clone https://github.com/Professor-G/AstroDynamics.git
    cd AstroDynamics
    pip install .

The modules can then be loaded directly:

.. code-block:: python

    from AstroDynamics import *
